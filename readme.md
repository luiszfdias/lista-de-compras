# 📝 Lista de Compras

> Uma aplicação web simples e responsiva para gerenciar sua lista de compras, desenvolvida com **HTML**, **CSS** e **JavaScript puro**.

## 🎯 Sobre o Projeto

Este projeto foi desenvolvido como parte do curso **Fullstack da Rocketseat**. Trata-se de um clone da tarefa de casa do curso, com foco na **manipulação de elementos do DOM via JavaScript**.

### O que aprendi

- ✅ Manipulação dinâmica de inputs e elementos do DOM
- ✅ Implementação de event listeners para interações do usuário
- ✅ Criação e remoção de elementos HTML através de JavaScript
- ✅ Validação e processamento de formulários
- ✅ Design responsivo com abordagem **Mobile First**

## 🎨 Características

- **Responsivo**: Funciona perfeitamente em dispositivos mobile, tablet e desktop
- **Mobile First**: Desenvolvido com prioridade para pequenas telas
- **Interface Limpa**: Design minimalista baseado no padrão da Rocketseat
- **Sem dependências externas**: Utiliza apenas JavaScript vanilla

## 🛠️ Tecnologias

- **HTML5** - Estrutura semântica
- **CSS3** - Estilos responsivos
- **JavaScript (Vanilla)** - Manipulação do DOM

## 🚀 Como Usar

### Clonar o projeto

```bash
git clone https://github.com/SEU_USUARIO/Lista-de-compras.git
cd Lista-de-compras
```

### Abrir no navegador

Você pode abrir o arquivo `index.html` diretamente no seu navegador.

## 📂 Estrutura do Projeto

```
Lista-de-compras/
├── index.html          # Estrutura HTML
├── scripts.js          # Lógica JavaScript
├── readme.md           # Este arquivo
├── assets/
│   └── icons/          # Ícones do projeto
└── styles/
    ├── global.css      # Estilos globais
    ├── index.css       # Importação dos arquivos de estilo
    ├── form.css        # Estilos do formulário
    ├── list.css        # Estilos da lista
    └── footer.css      # Estilos do footer
```

## 💡 Funcionalidades

- **Adicionar itens**: Digite um item e clique em "Adicionar item"
- **Marcar como concluído**: Use o checkbox ao lado do item
- **Remover itens**: Clique no ícone de lixeira para remover
